# Ship Observations Team (SOT) Website

Official website for the Ship Observations Team (SOT), coordinated under the World Meteorological Organization (WMO) and the Intergovernmental Oceanographic Commission (IOC-UNESCO).

The website presents the SOT mission, global ship-based observing networks, operational resources, events, and live observation maps powered by OceanOPS.

## Networks Covered

- Voluntary Observing Ship (VOS) Scheme
- Automated Shipboard Aerological Programme (ASAP)
- Ocean TraX (formerly XBT SOOP)

## Features

- Responsive multi-page static website
- Navigation menu with dropdown submenus
- Hero banner and statistics sections
- Informational content pages
- Events and resources sections
- Mobile-friendly navigation
- Custom styling with CSS and JavaScript
